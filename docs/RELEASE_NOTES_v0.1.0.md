# MenuForge v0.1.0

Initial MVP release.

## Included

- Spring Boot auto-configuration.
- JSON document storage.
- Public menu HTTP API.
- Optional API-key protected admin HTTP API.
- Internal Java services for custom website admin areas.
- Full menu import/export.
- Category, item, badge, allergen, tag and custom attribute model.
- Standalone admin demo app.
- Copy-friendly examples for real site integration.
- JSON Schema for the menu document.

## Not Included Yet

- Production admin UI.
- Login/roles.
- Image uploads.
- Telegram bot implementation.
- Hosted package registry publishing.
